Misc.
=====

Random Scripts for system and network administration

mssql_backup.py - copies SQL backup files to local backup repository and uploads to offsite backup location
remote-reboot.py - remotely reboot multiple servers for use in other scripts and scheduled tasks
rename-files.py - batch renaming of troublesome Sharepoint files
sftp-files.py - simpler version of mssql_backup.py, copies files to SFTP location
